To compile the code:

1) Run 'cmake ..' at `../Ruoyu_HW/bin`
2) Run 'make' at same dir

To run the code:

At same directory, run:
`./equation_evaluator /path/to/input.txt /path/to/output.txt`
Please make sure input file and output dir exist.
An example of output can be found at `../Ruoyu_HW/output/output.txt`
